※利用規約※

1. サービスの目的
　当サービスは、ユーザー同士の交流を目的としたオープンチャットです。


2. 禁止事項
　以下の行為は禁止します。見つけたら即BAN（アカウント停止）

　荒らし行為: 大量のメッセージ送信、画面を埋め尽くす行為、スパム、アカウント大量生産。

　迷惑行為: 他のユーザーへの誹謗中傷、脅迫、いやがらせ、アカウント乗っ取り

　犯罪行為: 法律に違反する行為、犯罪の予告・助長。

　個人情報の共有: 自分や他人の住所、電話番号などを載せること。

　その他: 管理者が「不適切」と判断した行為。

　当サービスを利用して発生した損害について、管理者は責任を負いません。
